# Writable ERDs

Every ERD that appears in the public Appliance API (`appliance_api.json`) and supports **write** operations according to the ERD definitions. The *Has Status ERD* and *Has Request ERD* columns indicate whether a complementary paired ERD exists, which can help identify how safe it is to write to that ERD.
